## Project Description
This project was based upon the development of a simple packet sniffing and network traffic analysis tool using Python. A packet sniffer is a program that captures network packets moving through a network interface. The project helped students practically understand important Computer Networking concepts such as IP addresses, MAC addresses, ports, protocols, packet size, TCP, UDP,ICMP, DNS, HTTP, HTTPS, and network traffic behavior.

## Tools and Libraries Used
* **Python 3**
* **Scapy** (for packet capturing)
* **Pandas** (for data processing)
* **Matplotlib** (for generating charts)
* **Pillow** (for automated screenshot captures)
* **Wireshark / Npcap** (for underlying network interfaces)
